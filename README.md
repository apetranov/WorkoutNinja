MVP Features:
1. Create workout - when creating workout user can add day of the week, muscles to hit in the workout and a timer for the length of the workout

2. Add exercise - when a workout has been created user can add exercises to workout. Each exercise user adds amount of sets and reps per set. Completed reps of set start at 0 and user can increment reps. When the reps reach the rep goal for the set the number of completed sets increments. After all the sets of a exercises have been finished exercise is marked as complete

3. Workout completed - after all of the exercises in a workout have been complete user is shown the amount of time it took to finish the workout
